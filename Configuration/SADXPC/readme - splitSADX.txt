Put these files in the folder with your hack's EXE.

To split data from the EXE, run "splitSADX.exe yourexe.exe splitSADX.ini".

To split data from DLL files for use with SADXLVL2, run "splitSADX.exe system\yourdll.dll yourdll.ini".

To put edited data into the EXE, run "buildSADX.exe yourexe.exe", and yourexe_edit.exe will appear in the folder.

ALWAYS split and build from a CLEAN file.
